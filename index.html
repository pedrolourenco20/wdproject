My design process

Before coming up with the HTML code for the portfolio, I started off by drawing a small sketch of how I visualised the website. I drew the prototype relatively simple, in order to allocate room for change and improvisation. The design was relatively accessible to come up with, so I would say the most difficult part of the project as a whole, would be implementing the JavaScript sections whilst making them functional. After finalising the design, I focused on building the structure and layout. I made sure to optimise the code for responsiveness and accessibility. The portfolio's main objective was to showcase my web development skills and design approach, and I feel that it turned out to be a fair representation of my abilities.
